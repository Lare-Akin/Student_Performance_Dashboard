This project is for a working prototype dashboard which users can use to analyse and gain insights into student performance
The whole project delivery as being shaped by understanding of users and their needs. 
A user centred design approach has been adopted - engaging with users to understand the problems to solved for them, testing the prototype with them to ensure their needs are met. 
Feedback for usability testing have been used for iterative development
Tis working prototype is still subject to continous iterative development because usability testing has only been done with 6 people. Best usability testing is done with someone knowing about the prototype for the first time

# Accessible Student Performance Dashboard

A modern, inclusive dashboard designed to help students, parents, teachers, and analysts understand the factors influencing academic performance.

##  Features
- Interactive filters for personalised insights
- Behavioural and socioeconomic analysis
- Accessibility‑focused design
- Auto‑generated insights for each user group
- Downloadable reports (CSV + ZIP)

## Tech Stack
- Streamlit
- Python
- Pandas
- Plotly
- Seaborn
